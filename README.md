# pvfParser

Node 22 + TypeScript PVF browser for the files under `fixtures/`.

## Workspace

- Root: `pnpm workspace`
- App: `apps/pvf-explorer`
- Runtime: Node.js native type stripping, no bundler / transpiler

## Commands

```bash
pnpm install
pnpm dev
pnpm check
pnpm test
```

The local explorer starts at `http://127.0.0.1:4318`.
